⚠️ 사용 지침
  • 키워드는 최대한 길고 구체적인 악성 표현으로 설정하세요.
  • 자동 차단 로그를 주기적으로 확인해 오탐된 계정을 복구해 주세요.
  • 평소 즐겨보거나 신뢰하는 채널은 미리 화이트리스트에 등록해 두세요.
Important Notice

Before using
Auto-block

This feature only checks for keyword presence — it cannot understand context.

⚠️ Guidelines
  • Set keywords as long, specific abusive phrases to reduce false positives.
  • Regularly review the auto-block log and recover any mistakenly blocked accounts.
  • Add channels you trust to the whitelist in advance.
개인정보 보호

데이터는
로컬에만

모든 차단 데이터는 사용자의 브라우저 로컬 저장소(내 컴퓨터)에만 안전하게 보관되며, 외부 서버로 절대 전송되지 않습니다.

'커뮤니티 평판' 기능은 기본 OFF이며, 직접 활성화한 경우에 한해 개인정보 없이 익명화된 데이터만 처리됩니다.

Privacy

Your data stays
local

All block data is stored securely and exclusively in your browser's local storage (your device). Nothing is ever sent to external servers.

The Community Reputation feature is off by default. When enabled, only anonymized data is processed — no personal information is collected.
